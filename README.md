# VR QA Test

- VR QA Test, using the BDD framework Cucumber to provide documentation to automated tests.

        $ git clone https://github.com/bnascimento89/vrTest

## Dependencies
 - Ruby 2.6.0

## Recommendations ##
 - Install bundler
         
         $ gem install bundler

## Project Installation

     $ bundle install
      
## Usage
 - run: 
     
       $ cucumber
